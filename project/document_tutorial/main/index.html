    Tài liệu hướng dẫn

    Trang web sử dụng mkdocs.org để dựng layout và dùng markdown làm ngôn ngữ mô tả.

    Hướng dẫn cài đặt#

    Cài đặt Python#

    • B1: Download python
    • B2: Cài đặt python như phần mềm bình thường

    Lưu ý: Khi cài đặt python phải chọn vào ô Add Python to Path như hình vẽ bên dưới

    • B3: Kiểm tra xem cài đặt đã thành công chưa bằng câu lệnh CMD
    $ python --version
Python 2.7.2
$ pip --version
pip 1.5.2
    • B4: Cài đặt gói bổ trợ
    $ pip install mkdocs
$ pip install mkdocs-material

    Chạy server#

    Sau quá trình cái đặt các bước trên hoàn tất, checkout code từ SVN về máy. Để chạy ứng dụng mở CMD lên và gõ lệnh sau:

    $ mkdocs serve

    Build ứng dụng#

    Để build ứng dụng lên bản online sử dụng câu lệnh CMD sau:

    $ mkdocs build

    Lúc đó dữ liệu sẽ được xuất vào thư mục site trong thư mục code

    Hướng dẫn viết bài#

    Get code từ SVN về: 

    https://192.168.1.240/svn/PHATTRIEN-TRIENKHAI/CQHC/Moi truong Phat trien/EFY-TUTORIAL/Code/efy-tutorial

    Cấu trúc thư mục trang tin bài docs#

    • img: File lưu ảnh
    • programming: thư mục lưu các bài viết về lập trình
    • project: Lưu các bài viết hướng dẫn về các dự án
    • software_dev: Các bài viết về công cụ lập trình, tool

    Cách thêm bài viết mới#

    • Xác định bài viết thuộc chủ đề nào
    • Tương ứng chủ đề định viết bài tạo file .md ở thư mục bài viết đó. VD: Viết hướng dẫn sử dụng cho Outlook thì vào thư mục software_dev tạo file outlook.md.
    • Cấu trúc viết hướng dẫn sử dụng ngôn ngữ Markdown để viết bài

    Một số chú ý khi viết bài#

    • Chú ý về các thẻ #: Các đề mục của bài viết nên bắt đầu từ ##, chỉ dùng # làm thẻ tiêu đề bài viết.

    • Commit nội dung bài viết lên máy chủ: Sau khi viết bài xong để commit bài viết lên máy chủ cần thực hiện các bước sau:

      B1: Built ứng dụng (Câu lệnh $ mkdocs build trong cửa sổ command line)

      B2: Commit dữ liệu lên máy chủ.

      B3: Chạy lại thử đường link online
